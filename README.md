# Smart Raz — אפליקציית למידה משחקית (גילאי 5–7)

Web App רספונסיבית (מובייל + דסקטופ) לתרגול **אנגלית**, **קריאה בעברית** ו**חשבון**, עם חוויה רגועה, חיזוקים חיוביים ועיצוב קו “חלל / מסע בין כוכבים”.

## טכנולוגיות

- **Next.js 16** (App Router) + **TypeScript**
- **Tailwind CSS v4**
- **ללא מסד נתונים ב-MVP** — התקדמות נשמרת ב־`localStorage` (מוכן להחלפה בשכבת שרת בהמשך)

## ארכיטקטורה (הפרדת תוכן ומנוע)

| שכבה | תיקייה | תפקיד |
|--------|--------|--------|
| תוכן לימודי | `content/` | בנקי שאלות לפי רמת קושי, מזהי משחקים, תיאורי מטרות |
| חוזה משחק (טיפוסים) | `lib/game-types/` | שאלות per-type, `registry`, `GameDefinition<Question>`, אדפטרים מלגאסי |
| מנוע משחק | `lib/game-engine/` | ערבוב אפשרויות, לוגיקת סיבובים, התאמות קושי בעת משחק |
| מצב שימוש | `lib/progress/` | כוכבים, רמה למדינה, זמן משוער, צילום למסך הורה |
| UI ילדים / הורים | `components/` | כפתורים גדולים, משוב, לייאאוט קוסמי |

## עקרונות UX (מוצר)

- מטרה אחת ברורה למסך; מעט טקסט; כפתורי מגע גדולים.
- שפה מעודדת בלבד (“כמעט”, “בוא ננסה יחד”, “רוצה רמז?”).
- משחקים קצרים (~5–8 דקות): סיבוב של **שאלות קבועות** עם נקודות כוכבים.
- רמות קושי: `gentle` → `steady` → `spark` (בתוכן ובהעדפות שמירה).

## מסלולים עיקריים

- `/` — מסך בית: שלושה “עולמות” (אנגלית, עברית, חשבון).
- `/world/[worldId]` — רשימת משחקים בעולם.
- `/play/[gameSlug]` — מסך משחק.
- `/parent` — אזור הורה (סיכום, זמן משוער, רמות).

## פיתוח

```bash
npm install
npm run dev
```

פתחו [http://localhost:3000](http://localhost:3000).

אם מופיע אזהרת `turbopack.root` בגלל `package-lock` כפול מחוץ לפרויקט, ב־`next.config.ts` מוגדר `turbopack.root` לתיקיית הפרויקט.

```bash
npm run build
npm run lint
npm run test
```

## המשך מומלץ

- ספריית משחקים נוספת עם אותו `GameModule`.
- עטיפת TTS לקריינות (`speechSynthesis` או שירות חיצוני).
- אימות הורים וסנכרון ענן.
